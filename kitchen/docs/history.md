# History

---

Matryoshka-Io originated as [Matryoshka](https://github.com/g41797/matryoshka){target="_blank" rel="noopener"}, a library written in Odin.

The Zig version is **not a transpilation**.  

It is a complete _redesign_

- include _integration_ with Zig Io. 

The name reflects that integration.

---

The result is **Matryoshka-Io**.

---

## Why Master?

---  
A matryoshka is a set of nested Russian dolls.

The term *Master* was inspired by the horror movie [The Doll Master](https://www.imdb.com/title/tt0416853/){target="_blank" rel="noopener"}.
